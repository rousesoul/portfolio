Restaurant Order Management System

A comprehensive, full-featured online order management system designed for the tourism and catering industry.

Project Overview

This project is a full-featured online order management system specifically designed for the tourism and catering industry. It aims to address the pain points of traditional order processing methods, such as inefficiency, lack of transparency, and proneness to errors. Through digitalization and automation, it helps catering businesses achieve fine-grained management of the entire business process chain, from customer ordering, order review, menu management, and price control to bill generation, data export, and multilingual support. The system not only improves operational efficiency and reduces management costs but also enhances the company's market competitiveness by optimizing the customer experience.

Core Value:

  • Improved Operational Efficiency: Automates order processing, reducing manual intervention, significantly increasing processing speed and accuracy.
  • Fine-grained Management: Provides comprehensive customer, menu, order, and financial data management, supporting multi-dimensional data analysis and export.
  • Enhanced Customer Experience: Customers can submit orders online, track status, and request changes with convenient and transparent operations.
  • Flexible Business Adaptation: Supports multi-role permission management, custom holiday service fees, future price settings, etc., to meet complex business scenarios.
  • Data-Driven Decisions: Offers rich report export functions to assist businesses in data analysis and decision-making.

Core Features & Functionality

User & Permission Management

Robust multi-role system (Admin, Sub-Admin, Customer with Operator/Financier/Head roles) ensuring data security and operational compliance via Role-Based Access Control (RBAC). Features secure registration, login, password hashing, and recovery, alongside account activation/deactivation and soft delete capabilities.

Image: User and permission settings interface.

Customer Relationship Management (CRM Lite)

Comprehensive management of tour company information, contact details, and account associations. Allows customers to submit profile change requests online, with admin approval workflows and full audit trails.

Dynamic Menu & Pricing Strategy

Supports menu categorization, item creation, editing, and soft deletion. Implements an intelligent pricing system including current/future pricing with automatic activation dates, differentiated pricing for adults, children (auto-calculated ratios), and staff (tour leaders, drivers, guides). Features flexible discount strategies like "1 free staff for every 9 customers" (configurable to include/exclude children).

Image: Menu items and pricing rules.

Core Order Management

Full lifecycle order processing: creation, editing, multi-level approval workflows (Pending, Approved, Rejected). Provides real-time order status tracking, intelligent payment management (status, date, auto invoice number generation, due dates), and online order change/cancellation requests with admin approval and history logging.

Image: Order list view with status indicators.

Powerful Data Export & Sharing

Multi-dimensional report exports: quick daily/monthly standard reports, and advanced custom exports with field selection and criteria filtering (date range, customer, payment status). Utilizes `XlsxWriter` for dynamic Excel generation with auto-formatting and smart column width adjustment. Secure, expiring, token-based encrypted links for sharing exported files.

Financial & Billing Management

Automated generation of professional invoices from order data, with online viewing and download. Integrated email notifications (Flask-Mail) for invoices, payment reminders, and order status changes. Customizable public holiday service fees automatically calculated and applied to orders and invoices.

Image: Example of a generated invoice or billing section.

System Configuration & Internationalization

Flexible admin settings for company info, default staff meal prices, SMTP server, notifications. Built-in multilingual support (Chinese, English) with real-time UI switching and persistent user preferences (Flask-Babel). Comprehensive database management: manual/auto backup, upload/restore, and recycle bin functionality.

Technical Deep Dive & Stack

1. Sophisticated Business Logic & High Configurability

  • Complex Discount Strategy Implementation: A core challenge was accurately implementing and configuring various discount logics, such as the "1 free for every 9 (including/excluding children)" staff discount. This required meticulous judgment and calculation of quantities, prices, and roles, achieved through flexible backend parameter configurations.
  • Dynamic Pricing System: Menu prices differentiate between current and future prices (with scheduled auto-activation) and are dynamically calculated based on customer type, party composition (adult/child/staff), and holiday service fees, ensuring pricing accuracy and business flexibility.
  • Multi-Role Permissions & Workflow: The system incorporates a deep understanding of tourism catering operations, featuring multi-level user roles and clear workflows for order approvals and information changes, ensuring accountability and standardized operations.

2. Powerful Data Processing & Interactive Experience

  • Intelligent Data Export & Excel Generation: The custom report export function allows users to select fields and apply multiple filter conditions. The backend uses `XlsxWriter` to dynamically generate complex Excel reports with intelligent column width adjustment and formatted output for specific data types.
  • Secure Temporary File Sharing: Addressed the need for sharing exported files by designing a token-based encrypted sharing link mechanism with expiration times for secure and controlled data sharing.
  • Asynchronous Tasks & Email Notifications: For operations like order submissions, status changes, and payment reminders, the system uses Flask-Mail integrated with Python threading (or APScheduler) for asynchronous email notifications, enhancing user experience by avoiding main process blockage.

3. Comprehensive System Management & Maintenance

  • Full Recycle Bin Mechanism: Core data like orders, customers, and menus feature soft-delete functionality with corresponding recycle bin management interfaces for data recovery or permanent deletion.
  • Automated Data Cleanup: APScheduler tasks automatically clean up recycle bin data and expired share tokens/files beyond a set period (e.g., 30 days), maintaining database cleanliness and system performance.
  • Robust Database Backup & Recovery: Supports manual and automated database backups (full or data-only). Backup files are storedサーバーサイド, with an interface for recovery, ensuring data disaster recovery capabilities.
  • Multilingual Internationalization (i18n & l10n): Full multilingual support (currently Chinese and English) via Flask-Babel for UI text, messages, and email content, with user preferences saved automatically.
